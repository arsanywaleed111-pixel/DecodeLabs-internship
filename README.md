# DecodeLabs-internship
هلاً بك في مستودع مشاريعي التقنية خلال فترة تدريبي في DecodeLabs. يعكس هذا المستودع تطوري في بناء الأنظمة الذكية، بدءاً من المنطق البرمجي وصولاً إلى تعلم الآلة.

📂 استعراض المشاريع
1. Simple AI Chatbot 🤖
الوصف: محرك محادثة بسيط يعتمد على القواعد البرمجية (Rule-based) لفهم مدخلات المستخدم والرد عليها.

الهدف: بناء أساسيات التفاعل البرمجي ومعالجة النصوص البسيطة.

2. Iris Flower Classifier 🌸
الوصف: نموذج تعلم آلي يستخدم خوارزمية K-Nearest Neighbors (KNN) لتصنيف أنواع زهور السوسن بناءً على خصائصها الفيزيائية.

الهدف: فهم أساسيات الـ Machine Learning، تقسيم البيانات، وتحجيمها (Scaling).

3. AI Recommendation Logic (Project 3) 🎯
الوصف: محرك توصية ذكي يطبق مفهوم "مواءمة الأنماط" (Pattern Alignment).

الهدف: استخدام خوارزمية Cosine Similarity لمطابقة اهتمامات المستخدم مع العناصر المتاحة، وهو المشروع الأساسي الذي يمثل مرحلة التخصص في الذكاء الاصطناعي.

🛠 التقنيات المستخدمة
اللغة: Python

المكتبات: Pandas, Scikit-Learn

المفاهيم: Logic Building, Pattern Matching, Similarity Metrics.
